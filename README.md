# IPL Data Analysis Project
Dataset Source: Kaggle IPL Dataset
## 📌 Overview
This project analyzes IPL match data to uncover insights about team performance, player statistics, and match outcomes.

## 🛠 Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
  
## 💪Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insights Generation
- Pandas Manipulation
  
## 📊 Analysis Done
- Team performance analysis
- Toss decision analysis
- Player of the Match analysis
- Match result analysis (runs vs wickets)
- Season trends

## 📈 Key Insights
- The dataset contains **74 rows and 20 columns**.
- **Gujarat Titans** won the most matches in the tournament.
- After winning the toss, most teams preferred to **field first**.
- **Kuldeep Yadav** and **Jos Buttler** won the most Player of the Match awards.
- Most matches were played at **Wankhede Stadium (Mumbai)**.
- **Yuzvendra Chahal** took the highest number of wickets in the tournament.
- **Chennai Super Kings** had the highest win margin (**91 runs**).
- **Jos Buttler** scored the most runs in the tournament (**651 runs**).
- The highest individual score was by **Quinton de Kock (140 runs)**.
- The best bowling performance was by **Jasprit Bumrah**.
